# **App Name**: K-Portfolio

## Core Features:

- Interactive Resume Parsing: Parse the information provided on the website, using AI as a tool, to personalize the user experience.
- Skills Showcase: Visually appealing display of technical skills (AI/ML, Full-Stack, etc.).
- Project Highlights: Interactive grid to showcase software projects with filtering.
- Contact Form with Validation: Form to enable easy and direct contact from potential employers or collaborators.
- Responsive Layout: Ensure website adaptability on different screen sizes (desktop, tablet, mobile).

## Style Guidelines:

- Background: Dark color scheme with a desaturated blue hue (#12182B) for a professional, tech-focused feel.
- Primary: Bright, vibrant blue (#64B5F6) to create contrast against the dark background, providing a clear focus for important interactive elements.
- Accent: A vivid cyan (#00BCD4), carefully calibrated in saturation and brightness compared to the primary color, intended to create high contrast and add sophistication.
- Font: 'Inter', a grotesque sans-serif font, is chosen for its clean and modern appearance suitable for both headlines and body text.
- Minimalist icons representing each skill category, ensuring they align with a clean design aesthetic.
- Use of a grid-based layout with clear sections for easy navigation and readability.
- Subtle animations on scroll and hover, adding interactivity without overwhelming the user experience.
